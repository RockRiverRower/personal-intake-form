# personal-intake-form

Phone-friendly capture surface for Scott's intake system. Posts to a Google Apps Script Web App that routes to the right backend (Tasks sheet, Calendar, Contacts, Master Lists, or `_obsidian_queue`).

- `index.html` — capture form (8 types)
- `dashboard.html` — read-only task dashboard

Working copies live in Drive at `2-WORKING/Personal Hub/Personal Intake Form/`. Push to this repo via `deploy_form.sh`.
